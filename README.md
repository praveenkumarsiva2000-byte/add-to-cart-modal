# Amart Cloth

## Project Description
This project fetches product data from Fake Store API and allows users to add products to cart using a modal interface.

## Features
- Fetch products from Fake Store API
- Add to cart functionality
- Alert if product already added
- Navbar cart count
- Modal displaying cart items
- Remove from cart option
- Fully responsive design
- Clean code structure

## Tech Stack
- ReactJS
- Tailwind CSS
- JavaScript
- HTML/CSS

## Local Setup (skip Netlify)
1. Open a terminal in the project root: `add-to-cart-modal`
2. Install dependencies:

```powershell
npm install
```

3. Start the dev server:

```powershell
npm start
```

4. Open http://localhost:3000 in your browser.
